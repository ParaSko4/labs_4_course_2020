New 13-inch MacBook Pro

The Apple MacBook Pro has long been an excellent option for professionals of all types, from creatives to those that need something for word processing and spreadsheets. This is the entry-level model of the MacBook Pro — but it still has a whole lot to offer.

The device has a 13-inch Retina display, which looks great for watching videos and editing photos. Not only that, but it’s pretty powerful under the hood, thanks to the Intel Core i5 processor, 8GB of RAM, and 128GB of storage. On the edges, there are a few Thunderbolt 3 ports — which can be used to charge the device, and for use with things like displays and other peripherals.

This model of the MacBook Pro usually comes at $1,299 — but it’s currently available for only $1,199, which is $100 off.

Acer Chromebook 11 N7

If you’re looking for an ultra-affordable laptop that you can use to browse the web and access Google apps and services, then this cheap Chromebook is the way to go. The laptop certainly isn’t very powerful, but it is nice and small, making it a great choice for taking on the go. 

Chromebooks in general are getting better and better, thanks to the fact that they can now run many Android apps. You’ll also be able to access services like Google Docs and Google Drive, which will be handy for many.

This Chromebook is usually available for $199.99 — but at $50 off, it’s only $149.99.

Apple 27-inch iMac

Perhaps you prefer a desktop to a laptop, and the iMac has long been a go-to for many users. This model of the iMac not only has a beautifully big 27-inch display, but it also boasts powerful specs under the hood, meaning that you can use it for all your productivity and creative needs.

The specs on offer by the iMac include a 6-core Intel Core i6 processor, along with a massive 1TB hard drive, which should be more than enough storage for the vast majority of users. As mentioned, the display on the iMac is nice and big — but it also has a hugely high resolution, coming in at 5K.

Normally, this model of the iMac is available for $1,799 — but you can currently get it for $1,599.99, which is almost $200 off.

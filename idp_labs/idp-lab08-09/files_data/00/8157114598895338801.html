ONU y Unión Europea promovieron una mediación para encontrar una salida a la crisis en Bolivia

El objetivo de ambas instancias es facilitar un acuerdo para renovar el tribunal electoral y que se convoquen elecciones.

18 Nov 2019 by Claudio Silva

Las Naciones Unidas (ONU) y la Unión Europea (UE) promovieron una mediación para buscar una salida a la crisis de Bolivia. Para ello convocaron a representantes del gobierno interino, del partido del expresidente Evo Morales y organizaciones civiles. El plan es un acuerdo para renovar el tribunal electoral y convocar elecciones.

El conflicto institucional, las movilizaciones que registran más de 20 personas fallecidas y la represión de los militares en las protestas, complican al país. El acercamiento parece la única vía para desbloquear la situación. "El diálogo es el camino apropiado para superar las diferencias entre bolivianos", expresó la Conferencia Episcopal.

La sucesora provisional de Morales no fijó una fecha para concretar las nuevas elecciones, aunque la constitución establece tres meses para tomar una decisión. Este lunes el gobierno interino no descartó tener que convocar los comicios con un decreto, como sugirió el principal rival de Morales en las elecciones de octubre, Carlos Mesa.
